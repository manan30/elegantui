# Input

Input component is used to get the user input in a text field.

## Usage

```jsx
import { Input } from '@elegantui/forms';
```

## Basic Example

<div style={{ marginTop: '1rem', width: '15rem' }}>
  <Input name='default' label='Input' placeholder='Enter text' />
</div>

```js
import { Input } from '@elegantui/forms';

export default function App() {
  return <Input name='default' label='Input' placeholder='Enter text' />;
}
```

### Variants

Input component is available in six variants.

<div
  style={{
    marginTop: '1rem',
    display: 'flex',
    flexDirection: 'column',
    gap: '0.5rem',
    width: '15rem'
  }}
>
  <Input
    name='primary'
    label='Primary'
    placeholder='primary variant'
    variant='primary'
  />
  <Input
    name='secondary'
    label='Secondary'
    placeholder='secondary variant'
    variant='secondary'
  />
  <Input
    name='error'
    label='Error'
    placeholder='error variant'
    variant='error'
  />
  <Input
    name='success'
    label='Success'
    placeholder='success variant'
    variant='success'
  />
  <Input
    name='warning'
    label='Warning'
    placeholder='warning variant'
    variant='warning'
  />
  <Input
    name='default'
    label='Default'
    placeholder='default variant'
    variant='default'
  />
</div>

```js
import { Input } from '@elegantui/forms';

export default function App() {
  return (
    <Input
      name='primary'
      label='Primary'
      placeholder='primary variant'
      variant='primary' // 'default' | 'secondary' | 'success' | 'warning' | 'error'
    />
  );
}
```

### Label

Input component renders a `label` for accessibility purposes. The value for the label can be provided using the `label` prop. If you would like to hide the label but still keep it accessible to screen readers you can pass a `hideLabel` prop.

import { Input } from '@elegantui/forms';

export default function App() {

    <Input
      name='primary'
      placeholder='primary variant'

### Custom styles

Input component allows you to configure your own styles. Styles can be provided either through the React `style` prop or CSS classes using `className` prop.

<div
  style={{
    marginTop: '1rem',
    display: 'flex',
    flexDirection: 'column',
    gap: '0.5rem',
    width: '15rem'
  }}
>
  <Input
    name='primary'
    placeholder='primary variant'
    variant='primary'
    label='Primary'
    style={{ backgroundColor: 'aliceblue', color: 'black' }}
  />
</div>

```js
import { Input } from '@elegantui/forms';

export default function App() {
  return (
    <Input
      name='primary'
      placeholder='primary variant'
      variant='primary'
      label='Primary'
      style={{
        backgroundColor: 'aliceblue',
        color: 'black'
      }}
    />
  );
}
```

#### Available Props

---

| **Prop**      | **Values**                                                                                      | **Default** |
| ------------- | ----------------------------------------------------------------------------------------------- | ----------- |
| **name**      | `string`                                                                                        | --          |
| **label**     | `string`                                                                                        | --          |
| **hideLabel** | <code>true &#124; false </code>                                                                 | `false`     |
| **variant**   | <code>default &#124; primary &#124; secondary &#124; error &#124; success &#124; warning</code> | `primary`   |
| **disabled**  | <code>true &#124; false</code>                                                                  | `false`     |
| **...rest**   | `React.InputHTMLAttributes<HTMLInputElement> `                                                  | --          |
