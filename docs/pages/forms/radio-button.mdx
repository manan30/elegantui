# Radio Button

Radio buttons allow users to select a single option from a list options. All possible options are exposed up front for users to compare.

## Usage

```jsx
import { RadioButton } from '@elegantui/forms';
```

## Basic Example

<div style={{ marginTop: '1rem', width: '15rem' }}>
  <RadioButton name='default' radioButtonText='Radio' value='radio' />
</div>

```js
import { RadioButton } from '@elegantui/forms';

export default function App() {
  return (
    <RadioButton name='default' radioButtonText='RadioButton' value='radio' />
  );
}
```

### Variants

RadioButton component is available in six variants.

<div
  style={{
    marginTop: '1rem',
    display: 'flex',
    flexDirection: 'column',
    gap: '0.5rem',
    width: '15rem'
  }}
>
  <RadioButton
    name='primary'
    value='primary'
    radioButtonText='Primary'
    variant='primary'
  />
  <RadioButton
    name='secondary'
    value='secondary'
    radioButtonText='Secondary'
    variant='secondary'
  />
  <RadioButton
    name='error'
    value='error'
    radioButtonText='Error'
    variant='error'
  />
  <RadioButton
    name='success'
    value='success'
    radioButtonText='Success'
    variant='success'
  />
  <RadioButton
    name='warning'
    value='warning'
    radioButtonText='Warning'
    variant='warning'
  />
  <RadioButton
    name='default-v'
    value='default'
    radioButtonText='Default'
    variant='default'
  />
</div>

```js
import { RadioButton } from '@elegantui/forms';

export default function App() {
  return (
    <RadioButton
      name='primary'
      value='primary'
      radioButtonText='Primary'
      variant='primary' // 'default' | 'secondary' | 'success' | 'warning' | 'error'
    />
  );
}
```

#### Available Props

---

| **Prop**            | **Values**                                                                                      | **Default** |
| ------------------- | ----------------------------------------------------------------------------------------------- | ----------- |
| **name**            | `string`                                                                                        | --          |
| **radioButtonText** | `string`                                                                                        | --          |
| **value**           | `string`                                                                                        | --          |
| **onChange**        | `() => void`                                                                                    | --          |
| **variant**         | <code>default &#124; primary &#124; secondary &#124; error &#124; success &#124; warning</code> | `primary`   |
| **disabled**        | <code>true &#124; false</code>                                                                  | `false`     |
| **className**       | `string`                                                                                        | --          |
