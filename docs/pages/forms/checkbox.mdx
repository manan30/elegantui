# Checkbox

Checkboxes allow users to select multiple items from a list of individual items, or to mark one individual item as selected.

## Usage

```jsx
import { Checkbox } from '@elegantui/forms';
```

## Basic Example

<div style={{ marginTop: '1rem', width: '15rem' }}>
  <Checkbox name='default' checkboxText='Checkbox' value='checkbox' />
</div>

```js
import { Checkbox } from '@elegantui/forms';

export default function App() {
  return <Checkbox name='default' checkboxText='Checkbox' value='checkbox' />;
}
```

### Variants

Checkbox component is available in six variants.

<div
  style={{
    marginTop: '1rem',
    display: 'flex',
    flexDirection: 'column',
    gap: '0.5rem',
    width: '15rem'
  }}
>
  <Checkbox
    name='primary'
    value='primary'
    checkboxText='Primary'
    variant='primary'
  />
  <Checkbox
    name='secondary'
    value='secondary'
    checkboxText='Secondary'
    variant='secondary'
  />
  <Checkbox name='error' value='error' checkboxText='Error' variant='error' />
  <Checkbox
    name='success'
    value='success'
    checkboxText='Success'
    variant='success'
  />
  <Checkbox
    name='warning'
    value='warning'
    checkboxText='Warning'
    variant='warning'
  />
  <Checkbox
    name='default'
    value='default'
    checkboxText='Default'
    variant='default'
  />
</div>

```js
import { Checkbox } from '@elegantui/forms';

export default function App() {
  return (
    <Checkbox
      name='checkbox'
      value='checkbox'
      checkboxText='Checkbox'
      variant='primary' // 'default' | 'secondary' | 'success' | 'warning' | 'error'
    />
  );
}
```

### Custom styles

Checkbox component allows you to configure your own styles. Styles can be provided either through the React `style` prop or CSS classes using `className` prop.

<div style={{ marginTop: '1rem' }}>
  <Checkbox
    name='primary'
    value='primary'
    checkboxText='Custom Styles'
    variant='primary'
    style={{ backgroundColor: 'chocolate', color: 'gray' }}
  />
</div>

#### Available Props

---

| **Prop**         | **Values**                                                                                      | **Default** |
| ---------------- | ----------------------------------------------------------------------------------------------- | ----------- |
| **name**         | `string`                                                                                        | --          |
| **checkboxText** | `string`                                                                                        | --          |
| **value**        | `string`                                                                                        | --          |
| **onChange**     | `() => void`                                                                                    | --          |
| **checked**      | <code>true &#124; false</code>                                                                  | `false`     |
| **variant**      | <code>default &#124; primary &#124; secondary &#124; error &#124; success &#124; warning</code> | `primary`   |
| **disabled**     | <code>true &#124; false</code>                                                                  | `false`     |
| **className**    | `string`                                                                                        | --          |
| **...rest**      | `React.InputHTMLAttributes<HTMLInputElement>`                                                   | --          |
