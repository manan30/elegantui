# Loading Spinner

Loading Spinner component allows users to get some indication when an async action is being performed. It can be used as a standalone component or inside any other container component.

## Usage

```jsx
import { LoadingSpinner } from '@elegantui/atoms';
```

## Basic Example

<div style={{ marginTop: '1rem' }}>
  <LoadingSpinner />
</div>

```js
import { LoadingSpinner } from '@elegantui/atoms';

export default function App() {
  return <LoadingSpinner />;
}
```

### Appearances

LoadingSpinner component is available in six appearances.

<div
  style={{
    marginTop: '1rem',
    display: 'flex',
    alignItems: 'center',
    gap: '0.5rem',
    overflowX: 'auto'
  }}
>
  <LoadingSpinner appearance='primary' />
  <LoadingSpinner appearance='secondary' />
  <LoadingSpinner appearance='error' />
  <LoadingSpinner appearance='warning' />
  <LoadingSpinner appearance='success' />
  <LoadingSpinner />
</div>

```js
import { LoadingSpinner } from '@elegantui/atoms';

export default function App() {
  return (
    <LoadingSpinner
      appearance='primary' // 'default' | 'secondary' | 'success' | 'warning' | 'error'
    />
  );
}
```

### Sizes

LoadingSpinner component is available in four sizes.

<div
  style={{
    marginTop: '1rem',
    display: 'flex',
    alignItems: 'center',
    gap: '0.5rem',
    overflowX: 'auto'
  }}
>
  <LoadingSpinner size='sm' appearance='primary' />
  <LoadingSpinner size='md' appearance='primary' />
  <LoadingSpinner size='lg' appearance='primary' />
  <LoadingSpinner size='xl' appearance='primary' />
</div>

```js
import { LoadingSpinner } from '@elegantui/atoms';

export default function App() {
  return (
    <LoadingSpinner
      size='sm' // 'md' | 'lg' | 'xl'
      appearance='primary'
    />
  );
}
```

### Custom styles

LoadingSpinner component allows you to configure your own styles. Styles can be provided either through the React `style` prop or CSS classes using `className` prop.

<LoadingSpinner
  size='sm'
  appearance='primary'
  style={{ marginTop: '1rem', color: 'chocolate' }}
/>

#### Available Props

---

| **Prop**       | **Values**                                                                                      | **Default** |
| -------------- | ----------------------------------------------------------------------------------------------- | ----------- |
| **appearance** | <code>default &#124; primary &#124; secondary &#124; error &#124; success &#124; warning</code> | `default`   |
| **size**       | <code>sm &#124; md &#124; lg &#124; xl </code>                                                  | `sm`        |
| **style**      | `React.CSSProperties`                                                                           | --          |
| **className**  | `string`                                                                                        | --          |
