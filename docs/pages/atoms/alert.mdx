# Alert

Alert allow users to be notified about certain events.

## Usage

```jsx
import { Alert } from '@elegantui/atoms';
```

### Default

<div style={{ marginTop: '1rem' }}>
  <Alert>Hello Friend</Alert>
</div>

```js
import { Alert } from '@elegantui/atoms';

export default function App() {
  return <Alert>Hello Friend</Alert>;
}
```

### Variant

Alert component is available in three variants

<div
  style={{
    marginTop: '1rem',
    display: 'flex',
    flexDirection: 'column',
    gap: '0.5rem',
    overflowX: 'auto'

}}

>

  <Alert variant='raised'>Raised</Alert>
  <Alert variant='flat'>Flat</Alert>
  <Alert variant='bordered'>Bordered</Alert>
</div>

```js
import { Alert } from '@elegantui/atoms';

export default function App() {
  return (
    <>
      <Alert variant='raised'>Raised</Alert>
      <Alert variant='flat'>Flat</Alert>
      <Alert variant='bordered'>Bordered</Alert>
    </>
  );
}
```

### Appearance

Alert component is available in six appearances

<div
  style={{
    marginTop: '1rem',
    display: 'flex',
    flexDirection: 'column',
    gap: '0.5rem',
    overflowX: 'auto'

}}

>

  <Alert appearance='primary'>Primary</Alert>
  <Alert appearance='secondary'>Secondary</Alert>
  <Alert appearance='error'>Error</Alert>
  <Alert appearance='warning'>Warning</Alert>
  <Alert appearance='success'>Success</Alert>
  <Alert>Default</Alert>
</div>

```js
import { Alert } from '@elegantui/atoms';

export default function App() {
  return (
    <>
      <Alert appearance='primary'>Primary</Alert>
      <Alert appearance='secondary'>Secondary</Alert>
      <Alert appearance='error'>Error</Alert>
      <Alert appearance='warning'>Warning</Alert>
      <Alert appearance='success'>Success</Alert>
      <Alert>Default</Alert>
    </>
  );
}
```

### Dismissble

Alert can be dimissed by adding a `dismissible` and `dismissHandler` prop. The `dismissHandler` is a callback function that is called when the `X` button inside the `Alert` is clicked

<div
  style={{
    marginTop: '1rem',
}}

>

  <Alert appearance='success' dismissible dismissHandler={() => {}}>
    Success
  </Alert>
</div>

```jsx
import { Badge } from '@elegantui/atoms';

export default function App() {
  return (
    <Alert appearance='success' dismissible dismissHandler={() => {}}>
      Success
    </Alert>
  );
}
```

#### Available Props

---

| **Prop**           | **Values**                                                                                      | **Default** |
| ------------------ | ----------------------------------------------------------------------------------------------- | ----------- |
| **children**       | `React.ReactNode`                                                                               | --          |
| **variant**        | <code>raised &#124; flat &#124; bordered</code>                                                 | `flat`      |
| **appearance**     | <code>default &#124; primary &#124; secondary &#124; error &#124; success &#124; warning</code> | `primary`   |
| **dismissible**    | <code>true &#124; false </code>                                                                 | `false`     |
| **dismissHandler** | `() => void`                                                                                    | --          |
| **customIcon**     | `React.ReactNode`                                                                               | --          |
| **className**      | `string`                                                                                        | --          |
