# ElegantUI Forms RadioButton component

## Installation

```bash
npm i @elegantui/forms
```

## Usage

```js
import { RadioButton } from '@elegantui/forms';

<RadioButton
  name='default'
  radioButtonText='Radio Button'
  onChange={eventCallback}
  value='default'
  disabled={disabled}
  variant={variant}
/>;
```
