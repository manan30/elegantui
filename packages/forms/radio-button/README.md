# ElegantUI Forms Checkbox component

## Installation

```bash
npm i @elegantui/forms
```

## Usage

```js
import { Checkbox } from '@elegantui/forms';

<Checkbox
  name='default'
  checkboxText='Subscribe to my newsletter'
  onChange={eventCallback}
  value='default'
  disabled={disabled}
  variant={variant}
/>;
```
