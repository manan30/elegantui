# ElegantUI Dialog component

## Installation

```bash
npm i @elegantui/atoms
```

## Usage

```js
import { Dialog } from '@elegantui/atoms';

<Dialog open>
  <Dialog.Title>Dialog Title</Dialog.Title>
  <Dialog.Content>
    <p>Add some content</p>
  </Dialog.Content>
</Dialog>;
```
